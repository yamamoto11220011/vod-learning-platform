# 🎥 VOD Learning Platform (vod-learning-platform)

### 🎯 What is it?
A video-on-demand e-learning platform where users can browse, watch, and track their progress on educational video courses. Built with Next.js 16 and Supabase, it provides a seamless and modern learning experience.

### ✨ Features
- **Modern Video Delivery**: Integrated `react-player` for smooth, responsive video playback.
- **Progress Tracking & Analytics**: Uses Supabase for robust real-time database management and user progress tracking.
- **Beautiful UI Components**: Styled with Shadcn UI and Tailwind CSS, featuring slick animations and accessible components.
